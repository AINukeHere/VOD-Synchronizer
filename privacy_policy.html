VOD Synchronizer 개인정보처리방침

최종 수정일: 2024년 9월 13일

요약: VOD Synchronizer는 사용자의 개인정보를 수집하지 않으며, 모든 데이터는 사용자의 브라우저에만 저장됩니다.

1. 개인정보 수집 및 이용

VOD Synchronizer는 사용자의 개인정보를 수집하지 않습니다. 본 확장 프로그램은 다음과 같은 데이터만 로컬 브라우저에 저장합니다:

  • 사용자 설정: 동기화 옵션, UI 설정 등
  • 타임스탬프 정보: VOD 동기화를 위한 임시 데이터
  • 마지막 동기화 정보: 사용자 경험 향상을 위한 로컬 캐시

2. 데이터 저장 및 처리

모든 데이터는 다음 방식으로 처리됩니다:

  • 로컬 저장: Chrome 브라우저의 로컬 스토리지에만 저장
  • 외부 전송 없음: 수집된 데이터를 외부 서버로 전송하지 않음
  • 암호화: 브라우저의 기본 암호화 기능 사용
  • 자동 삭제: 사용자가 언제든지 데이터 삭제 가능

3. 제3자 공유

VOD Synchronizer는 사용자 데이터를 제3자와 공유하지 않습니다:

  • 외부 서비스와의 연동 없음
  • 광고 네트워크와의 데이터 공유 없음
  • 분석 도구를 통한 데이터 수집 없음
  • 모든 기능은 확장 프로그램 내부에서 완결

4. 사용자 권리

사용자는 다음 권리를 가집니다:

  • 데이터 삭제: 언제든지 저장된 데이터 삭제 가능
  • 설정 변경: 개인정보 수집 관련 설정 변경 가능
  • 확장 프로그램 제거: 언제든지 확장 프로그램 제거 가능

5. 보안

VOD Synchronizer는 사용자 데이터 보호를 위해 다음 조치를 취합니다:

  • 최소한의 데이터만 수집
  • 로컬 저장소 사용으로 외부 접근 차단
  • 정기적인 보안 업데이트
  • 투명한 오픈소스 개발

6. 쿠키 및 추적 기술

VOD Synchronizer는 쿠키나 기타 추적 기술을 사용하지 않습니다.

7. 연령 제한

본 확장 프로그램은 모든 연령대의 사용자가 이용할 수 있습니다. 미성년자의 개인정보도 동일하게 보호됩니다.

8. 정책 변경

본 개인정보처리방침은 필요에 따라 변경될 수 있습니다. 중요한 변경사항이 있을 경우 확장 프로그램 업데이트를 통해 알려드립니다.
